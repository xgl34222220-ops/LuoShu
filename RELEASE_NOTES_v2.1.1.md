# 洛书 v2.1.1 预发行版

本版本继续深化 App 界面“所见即所得”的字体预览体验，不含字体引擎行为变更。

## 界面更新

- 字体选择弹窗：列表项图标块改为用该字体本身实时渲染（中文槽显示“中”、英文槽显示“Aa”、数字槽显示“123”），选字体时直接看到真身效果。
- 组合页：三槽结构概览与槽位卡片图标块，在已选字体后用所选字体实时渲染对应字符。
- Material 风格选择弹窗图标块配色由 tertiary 统一为 primary 色系，与全局一致。
- 无效字体全部保留静态文字兜底。

## 版本信息

- 模块版本：v2.1.1
- 模块 versionCode：20101
- 正式 App versionCode：2010101
- 发布渠道：GitHub 预发行版
