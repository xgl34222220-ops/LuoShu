# 洛书 v14.4.0

洛书 14.4 正式版完成 App-only 架构升级。发布内容只包含一个内置原生 App 的完整模块，不包含 WebUI，也不再提供 Lite 包。

## 更新内容

- 原生 App 字体库、字体组合、任务中心、导入与设置界面。
- 原生多选 ZIP 导入、持久化队列、恢复、暂停、继续、取消和失败重试。
- 导入前台服务、系统通知与任务中心联动。
- HyperOS 按真实 system、product、system_ext 分区映射字体，并保留原厂紧凑控件度量。
- 字体库直接应用支持静态多字重家庭和可变字体。
- 组合页读取字体自身默认轴，不再固定使用 wght 400。
- 未手动调整字重时，组合页默认生成 100 到 900 自动多字重。
- Miuix 液态玻璃界面、悬浮导入入口和任务状态视觉统一。
- 保持现有圆角、阴影、玻璃、颜色、布局和页面动画，同时优化重组、字体预览、缓存和后台任务性能。
- 正式 App 使用仓库固定签名，并启用 R8 代码优化与资源收缩。

## 版本

- 模块：v14.4.0
- 模块 versionCode：14400
- 正式 App versionCode：1440001
- 正式包名：io.github.xgl34222220.luoshu

升级时直接刷入模块并完整重启。模块内置 App 会按相同包名、固定签名和更高版本号执行更新。

应用自身内置字体或网页单独加载的字体不属于系统字体映射范围。不同 ROM 与 Root 管理器的挂载行为可能存在差异，刷入后请完整重启。
