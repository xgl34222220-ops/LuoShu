# 更新日志

## v14.1

- 新增完整中英数字体复合：中文为完整基底，仅导入英文和数字字形
- 所有 ROM 物理字体槽使用同一份完整字体，避免缺字回退差异和字符抢占
- 支持 glyf、CFF、CFF2、TTC 与常见可变字体
- 修复 glyf 字形边界缺失导致的 `xMin` 保存失败
- 生成流程改为 WebUI 主动后台任务，刷写和开机阶段不处理大字体
- 新增阶段进度、真实错误、事务提交、失败回滚与中断恢复
- 新增 SHA-256 组合缓存并限制最多三份
- 不覆盖系统字体 XML，不裁剪任何角色字体的字符覆盖
- 精简内置 ARM64 CPython 与 FontTools 运行时，降低包体积

## v14

- 轻量字体切换任务与 WebUI 状态恢复
- 文字、Emoji、Google/GMS 与微信 XWeb 兼容
- 支持 ColorOS 16、HyperOS 3、Android 16 与常见 Root 环境
