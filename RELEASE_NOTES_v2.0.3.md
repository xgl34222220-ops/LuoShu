# 洛书 v2.0.3 预发行版

本版本集中修复字体替换后的真实字形偏移、部分 Google 系应用字体不生效，以及重复切换字体耗时较长的问题，并继续完善多厂商与 Magic Mount 场景下的稳定性。

## 主要更新

- 修复换字体后文字抬高、页面整体偏移等问题，收敛字体 win metrics，并尽量对齐原厂 Roboto 度量。
- 新增 GMS Fonts Provider 缓存处理，使 Play 商店等 Google 系应用的输入框与页面字体能够跟随系统字体替换。
- 恢复系统字体或卸载模块时，会同步清理相关覆盖与缓存，避免残留字体继续生效。
- 增加字体归一化缓存，同一字体再次应用时可直接复用结果，明显缩短切换时间。
- 完善九档字重覆盖与复合字体生成逻辑，减少重复检测、重复生成和重复复制。
- 优化当前字体识别、历史任务与日志输出，减少无意义记录和状态噪声。
- 继续修复 Magic Mount RC 及多 OEM 分区环境下的字体任务兼容性。

这是预发行版本，建议重点验证：系统界面文字基线、Play 商店搜索框、九档字重、连续切换同一字体、恢复默认字体，以及重启后的字体持久化状态。发现问题时请附设备系统、Root 方案、字体文件和复现步骤。
