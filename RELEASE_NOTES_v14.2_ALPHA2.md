# 洛书 v14.2 Alpha2

本版本修正 Alpha1 字重调节只存在于预览页、无法参与字体组合的问题。

- 组合预设中的中文、英文和数字新增独立字重控制。
- 可变字体读取真实 `wght` 轴并显示连续滑块。
- 静态多字重字体只显示实际存在的 Thin / Light / Regular / Medium / SemiBold / Bold / Black 档位。
- 单一字重字体明确显示固定字重，不伪造滑动范围。
- 新增字体字重实例化器：将可变字体固定到所选轴位置，将 TTC/OTC 选择为最接近目标字重的字体面。
- 三个字重值进入任务状态、组合配置、缓存来源与最终复合字体生成链路。
- 继续复用 v14 已验证的完整复合、原子替换、ROM 映射、失败回滚和重启保护核心。

Alpha2 仍只对 `wght` 字重轴写入最终字体；`wdth`、`opsz`、`slnt` 等其他可变轴继续保留在工作台预览功能中。
