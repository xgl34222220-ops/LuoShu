# 洛书 v2.2.2

洛书 v2.2.2 完善了原生字体管理、组合预览、设备可信状态、备份归档和正式发布验证流程。本版本不改动字体生成、挂载、度量、事务和回滚核心逻辑。

## 主要更新

- 字体详情新增 SHA-256、稳定文件 ID、TTC/OTC 字体面、Family/Subfamily/PostScript、字重、字形与 Unicode 数量、CJK 覆盖、角色建议和可变轴。
- 首页新增设备字体可信状态、开机加载证据、脱敏诊断摘要、真机验收向导和匿名测试矩阵。
- 字体组合新增当前系统与候选方案 A/B 预览、四套快速字重方案、JSON 方案导入导出、Unicode 字形浏览，以及更紧凑的组合工具入口。
- 字体库新增 Family 分组、收藏、标签、批量整理、重复与命名冲突提示、SAF 字体目录监视、配置备份恢复和升级迁移检查。
- 支持导出真实 TTF、OTF、TTC 字体归档，并生成逐文件大小与 SHA-256 的 `manifest.json`；不会导出设备原厂字体或私人路径。
- 日志、字体库和组合页面统一处理底部安全区，修复工具按钮遮挡、重复底栏和大块无效留白。
- Root 不可用时显示明确中文引导，不再直接暴露 `Cannot run program "su"` 等底层异常。
- 新增正式更新元数据、预发行就绪报告、Android 编译诊断日志和完整候选成品校验。

## 安装

1. 在 Magisk、KernelSU、SukiSU Ultra 或 APatch 中刷入 `LuoShu-v2.2.2.zip`。
2. 完整重启设备；如果 Root 管理器提示其他 Meta 模块存在待生效变更，请先重启使其生效后再安装洛书。
3. 打开洛书并授予 Root 权限。

模块 ZIP 内置同版本正式签名 App。独立的 `LuoShu-App-v2.2.2.apk` 可用于单独安装或覆盖升级。
