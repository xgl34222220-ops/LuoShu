# 洛书 v2.2.1

洛书 v2.2.1 是单字体切换性能与状态提示热修复版，继续使用 v2.2.0 的设备自适应原厂字体清单和无 Hook 安全映射机制。

## 主要修复

- **减少大型字体重复读取**：同一份 CJK 字体映射到多个 ROM 槽位时，文件大小检查优先使用 `stat` 元数据，不再对硬链接别名逐个完整读取字体内容。
- **复用已验证字体结果**：App 完整验证后的字体按路径、设备号、inode、大小和时间戳记录身份；文件未变化时，后台切换任务复用验证结果，文件被替换或修改后自动重新完整验证。
- **保留性能门槛**：单字体切换继续使用 45 秒硬超时与事务回滚保护，没有通过放宽到 105 秒掩盖性能回退。
- **分阶段性能日志**：日志会分别记录验证、事务快照、槽位映射、负载校验、挂载同步和事务提交耗时，便于真机定位具体慢点。
- **修复失败提示样式**：字体切换失败或回滚时显示错误提示，不再出现绿色成功勾与失败文字同时显示的矛盾状态。

## ROM 与真机扫描

所有 ROM 仍优先读取当前设备生成的原厂字体清单，按实际分区、字体 XML、真实路径、TTC index 和度量映射。ColorOS、HyperOS 与其他品牌均不是固定槽位白名单；只有本机清单不可用时才使用静态回退规则。

## 安装与升级

刷入 `LuoShu-v2.2.1.zip` 后完整重启。模块内置固定签名正式 App，也可以单独安装 `LuoShu-App-v2.2.1.apk`。应用或恢复字体完成后仍需完整重启，避免系统字体缓存和挂载状态不一致。
