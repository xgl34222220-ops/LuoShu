# 洛书 v14

本次正式版重点解决字体切换体验、权限提示和 WebUI 流畅性问题。

- 修复字体切换期间 WebUI 偶发退出。切换状态改为读取轻量任务文件，不再高频重复扫描和重建字体预览文件。
- 切换任务会保存到本地；中途退出或 WebUI 被系统回收后，重新进入会自动确认切换结果。
- WebUI 核心脚本统一通过 `sh` 调用，开机自动静默校正权限，不再需要手动点击“修复脚本权限”。
- 移除用户侧“脚本权限”检查、修复按钮和“上一个稳定配置”回滚入口。
- 模块说明精简为“Android 全局字体管理，当前字体：xxx”，切换完成后自动更新字体名称。
- 重新精修卡片层级、圆角、阴影、交互反馈和小屏布局，减少大面积模糊与滚动渲染开销。
- 字体卡片使用按需渲染和布局隔离，字体数量较多时滚动更流畅。
- 保留一次开机一次安全切换、完整重启保护、ZIP 安全导入、可变字体、静态多字重、Emoji、GMS/Gemini/XWeb 和 Mountify 适配。
