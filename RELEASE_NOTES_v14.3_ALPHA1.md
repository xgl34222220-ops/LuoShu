# 洛书 v14.3 Alpha1.6

## 白泽 Miuix 视觉统一

- 本次版本只重做原生 App UI，不新增字体功能，也不修改字体切换、复合生成、导入或系统粗细写入逻辑。
- 首页、字体库、字体工作台和运行日志统一采用白泽当前的 Miuix 视觉系统：大标题、克制的渐变背景、实心超椭圆卡片、紧凑分组列表与清晰的信息层级。
- 底部导航改为白泽同款液态悬浮栏，图标位于文字上方，使用滑动选中指示器，并加入淡入与水平滑动页面转场。
- Android 12 及以上使用系统动态色，深色模式保持与白泽一致的深色背景和卡片层级。

## 页面调整

- 首页以当前字体和模块状态作为主视觉，Root、挂载引擎、字体库与工作台入口改为白泽式状态卡和分组行。
- 字体库改为统一的实心卡片列表，保留真实字体预览、直接应用、删除与系统字体恢复。
- 字体工作台重新排版中文、英文、数字槽位、真实可变轴、静态多字重、覆盖诊断与最终应用按钮，但功能行为保持 Alpha1.5 不变。
- “字重偏移”不再作为主要术语展示，界面统一使用“全局粗细微调”“向左更细、向右更粗”。
- 日志页、确认弹窗与空状态统一使用同一套圆角、间距和文字层级。

## 保留能力

- 保留真实可变字体多轴实时预览、静态多字重真实文件切换与完整轴参数提交。
- 保留同一字体且为标准 Regular 400 时直接应用。
- 保留系统全局粗细微调与中文、英文、数字、标点覆盖诊断。
- 保留 TTF、OTF、TTC 与字体模块 ZIP 的安全导入。

本版本为 UI 真机测试版，PR 继续保持 Draft，不合并、不发布。
