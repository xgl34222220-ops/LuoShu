# 洛书 v14.3 Alpha1

v14.3 Alpha1 开始完善原生 App 的字体接入体验。本阶段首先完成真正可用的系统文件选择器导入链路，不再要求用户手动把文件复制到固定目录。

- 可一次选择多个 TTF、OTF、TTC 或字体模块 ZIP。
- 所选文件只会先进入洛书 App 私有缓存，再由 Root 桥读取。
- Root 桥只接受洛书正式版或 Debug App 的受信任缓存路径。
- 单个文件限制为 256 MB，字体会检查真实格式并拦截无效或不安全字体。
- 字体按 SHA-256 去重；同名但内容不同的文件使用短哈希区分，不覆盖已有字体。
- 字体模块 ZIP 继续通过安全导入器处理，只提取字体文件，不执行包内脚本。
- 导入完成后自动刷新字体库，并分别报告成功、重复和失败数量。

Alpha1 后续提交将继续实现稳定字体唯一 ID、同名字体区分、TTC 字体面选择和字体详情页。本版本仍为开发测试版，不建议作为最终稳定版本长期使用。
