# 洛书 v2.0.4 预发行版

本版本主要解决升级后仍复用旧字体度量负载、导致 v2.0.3 的文字基线修复未实际生效的问题，同时补齐 mix 复合字体模式下的 Google 系应用字体覆盖，并整体升级缓存代际。

## 主要更新

- 字体负载架构升级至 v3，自动识别并强制重建旧度量负载，避免升级后继续复用旧字体数据。
- 首次开机可在后台完成旧负载重建，并通过通知提示用户在完成后重启，使新度量规则真正生效。
- mix 复合字体链路接入 GMS Fonts Provider 缓存处理，Play 商店搜索框等 Google 系应用在 mix 模式下也能跟随系统字体。
- 复合字体、多字重及相关任务缓存代际统一递增，防止旧度量缓存被错误复用。
- 同步调整模块更新状态与安全检查逻辑，并更新对应自动化测试，提升升级路径的稳定性。

## 测试提示

这是预发行版本。由 v2.0.3 或更早版本升级后，请重点观察首次开机的后台重建提示，并按提示重启。建议验证系统界面文字基线、Play 商店搜索框、mix 复合字体、多字重模式、重复切换字体以及恢复系统字体后的状态。反馈问题时请附设备系统、Root 方案、升级前版本、字体文件和完整复现步骤。
