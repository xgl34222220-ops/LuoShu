# 洛书 v14.2 Alpha6

Alpha6 先收口发布与安全基础，不改动已验证的 ROM 挂载架构。

- 版本统一由 `module.prop` 驱动，模块、App、WebUI 与产物名称不再各写一份。
- 正式 Release 仅发布固定密钥签名的 App，不再把临时 debug 签名当正式版本。
- Release 只接受已存在且与版本匹配的 Tag，并禁止覆盖同名版本。
- 同时提供 Full（内置 App）与 Lite（不内置 App）模块包。
- 全局应用字体前检查常用中文、英文、数字和标点覆盖，降低方框乱码与应用崩溃风险。
- CI 缓存 ARM64 字体运行时，减少重复构建时间。

> Alpha6 仍为测试版。刷入前请保留可用模块备份，并在 ColorOS 16 / HyperOS 3 真机完整重启后验证。
