# 洛书 v2.0.6 预发行版

本版本继续修复文字显示问题：在 v2.0.5 解决"字体被静默重置回默认"之后，部分应用（使用 includeFontPadding=false 渲染）仍出现标题与副标题重叠、小标签文字被裁掉一截的问题。

## 主要更新

- 字体度量契约修订：hhea ascent/descent 现在包住字形真实轮廓极值（下限保持 0.928/0.244 原厂契约），includeFontPadding=false 的应用不再出现重叠或裁切。
- typo 与 win 度量保持契约不变，includeFontPadding=true 应用的行距与基线完全不受影响，不会重新引入"文字抬高"。
- 归一化缓存版本升至 v3，字体负载架构升级至 v4：升级后首次开机自动后台重建负载，重建失败保留当前字体并下次开机重试。
- 复合字体与多字重缓存代际全量递增，杜绝旧度量缓存复用。

## 测试提示

这是预发行版本。升级后请在洛书 App 内重新应用一次字体并完整重启。建议重点验证：酷安话题页标题与热度、QQ 联系人年龄标签与输入框、系统时钟数字与英文、Play 商店搜索框，以及连续多次重启后的字体保持情况。反馈问题时请附设备系统、Root 方案、升级前版本、字体文件和完整复现步骤。
