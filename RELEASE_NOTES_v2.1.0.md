# 洛书 v2.1.0 预发行版

本版本聚焦 App 界面美化与视觉一致性优化，不含字体引擎行为变更。

## 界面更新

- 字体库卡片左上角 "Aa" 图标改为用字体本身实时渲染（可变字体按 400 字重），划过去一眼认出字体气质；无效字体保留静态兜底。
- Material 风格底部导航新增滑动选中指示器（弹簧动画），与 Miuix 风格一致的切换手感。
- 补全 Material 与 Miuix 两套排版的 headlineSmall / titleSmall / bodyLarge / bodySmall / labelSmall 档位，全 App 文字层级更统一，不再回退到系统默认字号。

## 版本信息

- 模块版本：v2.1.0
- 模块 versionCode：20100
- 正式 App versionCode：2010001
- 发布渠道：GitHub 预发行版

## 测试提示

这是预发行版本。建议两种界面风格（Material / Miuix）都切换看看：字体库卡片预览、底部导航切换动画、深浅色模式下的文字层级。反馈问题请附设备系统、Root 方案与截图。
