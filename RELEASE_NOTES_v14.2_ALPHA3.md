# LuoShu v14.2 Alpha3

## 重点修复

- 修复工作台点击“生成并应用”时，三份字体实例化在 WebUI Shell 调用中同步执行导致界面卡死的问题。
- 多轴实例化与完整复合任务全部转入后台，前端立即获得任务 ID 并只轮询状态与进度。
- 修复工作台组合卡片被旧三列网格压缩为约三分之一宽度的问题，移动端改为中文、英文、数字三个全宽卡片。
- 概览组合区新增与工作台共用的字体轴控件，不需要再进入单独页面调节。
- 收紧 MutationObserver 范围，避免控件重绘触发自身再次重绘。

## 完整可变轴

- 可变字体会显示其真实 `fvar` 轴，包括 `wght`、`wdth`、`opsz`、`slnt`、`ital`、`GRAD` 及字体自定义轴。
- 三个字体槽分别保存轴值，所有存在于源字体中的轴都会固定进最终实例字体。
- 静态多字重字体仍只允许选择实际存在的 Thin、Light、Regular、Medium、SemiBold、Bold、Black 文件。
- 单一静态字重明确显示固定状态，不伪造连续调节能力。

## 兼容与安全

- 继续复用 v14 已验证的完整复合、ROM 映射、原子提交、失败回滚与重启保护。
- 轴参数进入任务状态、组合配置和最终字体哈希，避免不同轴值错误复用同一缓存。
