# 洛书 v2.0.3-rc1

这是 v2.0.3 的首个候选版本，重点验证字体真实轮廓、混排基线、ROM 直连字体槽、后台任务和多挂载方案的一致性。

- 统一中文、英文与数字的真实轮廓、基线和垂直偏移，继续修正输入框、标题、桌面名称与混排页面的视觉错位。
- 补齐 ColorOS Google Play 搜索框等直连字体槽，并按真机位置覆盖 system、product、system_ext、my_product 等分区。
- 九档字重继续使用真实静态字重文件或可变字体实例，不以只修改字重元数据代替轮廓生成。
- 减少重复字体校验、生成、复制和负载收尾操作，同时保留完整事务校验、启动保护与失败回滚。
- 修正当前字体状态误显示为系统默认、历史任务残留常驻和无效日志噪声。
- 完善 Magic Mount RC/RS 与多 OEM 分区适配，保持 Magisk、KernelSU、SukiSU Ultra、APatch、Mountify 等环境的安全切换逻辑。
- 完整保留 v2.0.2 的正式 App 覆盖升级、刷写即时解除遗留 `disable` 和明确字体事务重新启用模块的修复。
- 模块版本为 `v2.0.3-rc1 / 20003`，正式 App `versionCode` 为 `2000301`。

该版本为候选测试包，不作为最终稳定版。请重点回归 HyperOS 3 与 ColorOS 16 上的桌面名称、输入框、Google Play 搜索框、九档字重、字体组合任务恢复和完整重启后的真实挂载结果。
