# 洛书 v14.1.1 RC2

## 重点变化

- 保留 v14.1 / Alpha10 已验证的完整复合字体算法。
- 修复复合任务真实阶段、百分比、缓存命中和完成状态显示。
- 字体负载与配置状态共同参与事务回滚。
- 防止多个 WebUI 页面重复启动同一复合任务。
- 运行时自检同时校验启动器、libpython、复合引擎和 FontTools。
- 清理旧版、孤立和临时复合缓存。

## 功能范围清理

- 彻底移除 Emoji、图标字体替换的安装、后端、WebUI、开机状态和 ZIP 导入功能。
- 升级时清理模块内部的 `NotoColorEmoji*` 负载和 Emoji 状态。
- 不删除用户公开存储中的个人文件。
- 保留图标字体、彩色字体和符号字体识别，仅用于防止误当作文字字体导入。
