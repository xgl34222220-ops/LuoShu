# 洛书 v14.2 RC2.1

RC2.1 是 v14.2 RC2 的字体角色门禁修正版，重点修复部分上一版本可正常使用的中文基底被误判为缺少标点的问题，不改变已经验证的字体挂载与事务提交架构。

- 统一多轴任务、缓存与内部字体名称，清理实验阶段遗留命名。
- 原生 App 启动后可重新接管正在运行或已经完成的字体任务。
- 待重启状态由模块持久保存，关闭或重开 App 后不会丢失。
- 可变字体按自身真实字重最小值、默认值和最大值选择。
- 中文基底只检查必要中文、英文字母和数字槽位；英文与数字角色只检查真正必需的字形。
- 半角标点恢复为可选替换项，不再误拦截花轮丸等可正常参与组合的字体。
- 安装脚本只部署模块并清理历史状态，不在刷写阶段生成字体、结束后台进程或直接安装 App。
- 终端入口只保留状态、列表、应用、恢复默认和日志，不再暴露上一字体回滚、热刷新或重启 SystemUI。
- WebUI 不再显示“仅重启系统界面”入口，字体变更统一要求完整重启。
- 发布流程在构建签名产物前强制执行源码、原生 App、模块包、角色门禁和稳定性检查。

RC2.1 仍为预发行测试版。刷入后必须完整重启，并确认 App 显示 `v14.2 RC2.1`。正式发布前还需完成 ColorOS 16、HyperOS 3、原生挂载与 Mountify 或 Hybrid Mount 的真机回归。
