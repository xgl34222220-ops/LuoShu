# 洛书 v14.2 RC2

RC2 是 v14.2 正式版前的第二个稳定候选版本，重点修复 RC1 原生 App 和多轴任务链路的状态一致性问题，不改变已经验证的字体挂载与事务提交架构。

- 统一多轴任务、缓存与内部字体名称，清理实验阶段遗留命名。
- 原生 App 启动后可重新接管正在运行或已经完成的字体任务。
- 待重启状态由模块持久保存，关闭或重开 App 后不会丢失。
- 可变字体按自身真实字重最小值、默认值和最大值选择。
- 发布流程在构建签名产物前强制执行源码、模块包和稳定性检查。
- 发布说明按当前版本自动选择。

RC2 仍为预发行测试版。正式发布前必须完成 ColorOS 16 与 HyperOS 3、原生挂载与 Mountify 或 Hybrid Mount 的真机回归，并验证 Google Play、微信 XWeb、失败恢复和两次完整重启。
